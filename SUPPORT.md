# Support

## Where To Ask What

- Bug report: open a GitHub issue with the bug template
- Feature request: open a GitHub issue with the feature template
- Security issue: follow [SECURITY.md](SECURITY.md)
- Development question: include platform, Rust version, command output, and the exact step that failed

## What Helps Maintainers Triage Faster

- Operating system and version
- `rustc --version`
- The exact command you ran
- Expected behavior
- Actual behavior
- Logs, screenshots, or terminal output when relevant

## Scope

Maintainers can usually help with:

- Reproducing bugs in the CLI or desktop app
- Build and test failures in supported environments
- Questions about architecture and contribution workflow

Maintainers may decline:

- Long-running private support
- Bespoke consulting
- Urgent response guarantees
